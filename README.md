# PopularMovies
Shows popular movies using themoviedb.org REST API using Retrofit, RxJava,Glide, MVVM and Pagination.

## Skills
Android, Kotlin, Retrofit, RxJava,MVVM, Glide and Pagination

## Widgets
Recycler View, Grid Layout
